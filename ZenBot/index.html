ZenBot© was founded in 2023 by MXB (also known as Maxim Bogdan). First it was a simple bot with simple commands like /hey which responded with "Hi!" or /ping which responded with "Pong!".
After a couple months it was able to do simple calculations like adding (with /add), subtracting (with /subtract), multiplication (with/multiply), division (with /divide) and even powers (with /pow).
Soon it will be able to make embeds, , and many other
It was created with the VSCode IDE using the Discord.js library in JavaScript.
